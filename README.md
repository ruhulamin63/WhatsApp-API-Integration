# WhatsApp-API-Integration
How to Send WhatsApp Messages With Laravel 11?

```bash
git clone https://github.com/ruhulamin63/WhatsApp-API-Integration.git
```

```bash
cp .env.example .env
```

```bash
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=your-db-name
DB_USERNAME=root
DB_PASSWORD=
```

```bash
composer update
```

```bash
php artisan migrate
```

### Public Access Route
```bash
http://127.0.0.1:8000/whatsapp
```
